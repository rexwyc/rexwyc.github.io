How to Play

  • Click on a block to select it and all adjacent blocks of the same color.
  • You can only blast groups of 2 or more connected blocks.
  • Larger groups give more points (2x for 4 blocks, 3x for 6 blocks, etc.).
  • When blocks are removed, blocks above will fall down to fill empty spaces.
  • The game ends when there are no more groups of 2+ blocks left.
  • Try to clear as many blocks as possible for a high score!
