Rules & Adjective Order

In English, adjectives usually follow a specific order when multiple adjectives describe the same noun:

  • Opinion (beautiful, ugly, nice)
  • Size (big, small, tall)
  • Age (old, young, new)
  • Shape (round, square, rectangular)
  • Color (red, blue, green)
  • Origin (American, Chinese, Victorian)
  • Material (wooden, plastic, silk)
  • Purpose (sleeping, cooking, racing)
Order: Opinion → Size → Age → Shape → Color → Origin → Material → Purpose
